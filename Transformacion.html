Para transformar la ecuacion Ax + By + C=0 a la forma x/a + y/b=1,se calcula la ordenada y la absisa al origen con las siguientes formulas:
a=-c/A----->Abscisa al Origen

b=-c/B----->Ordenada al Origen

Y se sustituye en la ecuacion simetrica de la recta:

Ejemplo1:

Transformemos a su forma simetrica la siguiente ecuacion:

X + Y - 4 = 0


/center>
Para ello asignamos: A=1, B=1, C=-4
Ahora sustituimos:
a = -c/A, a=-(-4)/1 = 4/1 = 4
b = -c/B, a=-(-4)/1 = 4/1 = 4

Nos quedaria como resultado:

x/4 + y/4 =1----->ecuacion simetrica

Ejemplo2:

Transformemos a su forma simetrica la siguiente ecuacion:

2x - y = 0


/center>
Ahora sustituimos:
a = 0/2 = a = 0
b = 0/1 = b = 0

Nos quedaria como resultado:

x/0 + y/0 =1----->ecuacion simetrica
